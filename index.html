Features

Basic Operations

  • Read data from magnetic stripe cards
  • Write data to magnetic stripe cards
  • Copy data between cards
  • Erase tracks selectively

Batch Processing

  • Read multiple cards to one file
  • Write multiple cards from one file
  • Export data files to iCloud
  • Import data files from storage

Technical Features

  • Support 210 BPI and 75 BPI
  • Adjustable leading zero
  • ISO data format support
  • AAMVA & CA DMV format (Coming soon)

Compatibility

  • Works with MSRX6/MSRX6BT device
  • iOS and Android support
  • Bluetooth connection
  • Compatible with Deftun/Misiri/Postech
